Avaliação Inetz

1. Sobre o modelo cliente servidor, podemos afirmar que:

  • A comunicação é iniciada pelo servidor
  • O servidor é responsável por servir dados aos clientes
  • O cliente é quem consome recursos do servidor
  • As respostas são originadas pelo cliente
Palavra: O cliente solicita e o servidor provê o recurso.

2. Um servidor de aplicação como uvicorn:

  • Somente em produção
  • Pode servir em loopback
  • Pode servir em rede local
Palavra: Suporta interfaces locais e externas.

3. O que significa URL?

  • Localizador de Recursos Locais
  • Localizador Uniforme de Recursos
  • Rotas Locais Unidas
  • Uma Rota Local
Palavra: Uniform Resource Locator.

4. Qual dessas opções faz parte da URL:

  • Caminho
  • Protocolo
  • Porta
  • HTML
  • Verbo
  • Endereço
Palavra: Protocolo, Endereço, Porta e Caminho.

5. Qual desses campos NÃO faz parte do cabeçalho HTTP?

  • Accept
  • Content-Type
  • Corpo
  • Server
Palavra: O Corpo é separado do cabeçalho.

6. O verbo PUT tem a função de:

  • Atualizar um recurso existente
  • Solicitar um recurso
  • Todas as anteriores
  • Deletar um recurso
Palavra: PUT serve para atualizações completas.

7. Respostas com códigos da classe 500, significam:

  • Erro no cliente
  • Informacionais
  • Erro no servidor
  • Sucesso
Palavra: Erro interno do servidor.

8. O código 200 de resposta significa:

  • OK
  • Created
  • Found
  • Accepted
Palavra: Requisição processada com sucesso.

9. O código 422 de resposta significa:

  • Bad Request
  • Forbidden
  • Not Found
  • Unprocessable Entity
  • OK
Palavra: Erro de validação de dados.

10. Qual a função do pydantic?

  • Documentação
  • Validar saída
  • Validar entrada
Palavra: Validação rigorosa de dados.
